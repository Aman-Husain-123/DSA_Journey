## üìñ How to Use

1. Clone the repository:
```bash
git clone https://github.com/Aman-Husain-123/DSA_Journey.git
cd DSA_Journey
```

2. Set up a virtual environment (optional but recommended):
```bash
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
```

3. Install Jupyter Notebook:
```bash
pip install jupyter
```

4. Launch Jupyter Notebook:
```bash
jupyter notebook
```

5. Navigate to any topic folder and open the `.ipynb` files to explore implementations and solutions.
